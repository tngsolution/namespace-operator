# Security Policy

## Supported Versions

The following versions are currently supported with security updates:

| Version | Supported |
|---------|-----------|
| >= 1.3.x | ✅ |
| < 1.3.0  | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately by:

- Opening a GitHub Security Advisory (recommended)
- Or emailing: security@tngsolutions.fr (if applicable)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)

We aim to respond within **72 hours**.

## Disclosure Policy

- Vulnerabilities will be assessed and validated.
- A fix will be prepared before public disclosure.
- Security releases will be clearly documented in the changelog.

Thank you for helping improve the security of this project.
